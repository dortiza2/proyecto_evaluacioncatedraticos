## What is Easy Docs?

Easy Docs is a simple and modern documentation management app built using [Fumadocs](https://github.com/fuma-nama/fumadocs) and [Next.js](https://github.com/vercel/next.js). It helps developers create and manage documentation with ease.

## Features

- MDX support for rich documentation content.
- Powerful layout and UI components from Fumadocs.
- Responsive design and custom styles.

## Getting Started

To set up Easy Docs locally, follow these steps:

<Steps>
  <Step>
    <span>Clone the repository</span>
    <span>Use `git clone` to copy the project to your local machine.</span>
  </Step>
  <Step>
    <span>Install dependencies</span>
    <span>Run `npm install` to install required packages.</span>
  </Step>
  <Step>
    <span>Start the development server</span>
    <span>Use `npm run dev` to launch the app locally.</span>
  </Step>
</Steps>
